# 🎧 Mood-Based Study Music Recommender

An AI-powered music recommendation system that suggests the most suitable study music based on a student's mood, task type, and energy level.

---

## 📌 Overview

Choosing the right music while studying can improve focus, concentration, and productivity. This project uses Machine Learning to recommend personalized study music categories based on user preferences and current state of mind.

Users provide:

- Mood
- Task Type
- Energy Level

The system predicts the most suitable music category for that situation.

---

## ✨ Features

- Mood-based recommendations
- Task-specific music suggestions
- Energy level analysis
- Machine Learning prediction
- Interactive user interface
- Fast recommendations
- Student productivity focused

---

## 🎯 Problem Statement

Students often struggle to find the right type of music while studying.

Different study activities require different environments:

- Coding may require deep focus music
- Reading may require calm ambient sounds
- Exam preparation may require distraction-free audio

This project provides personalized recommendations to help students stay productive.

---

## 🧠 Machine Learning Approach

The project uses supervised machine learning to learn patterns between:

- Mood
- Task Type
- Energy Level

and predicts the most suitable study music category.

### Example Inputs

| Mood | Task | Energy |
|--------|--------|--------|
| Happy | Coding | 8 |
| Stressed | Reading | 3 |
| Motivated | Exam Prep | 9 |

### Example Outputs

| Recommendation |
|---------------|
| Deep Focus Electronic |
| Nature Sounds |
| Ambient Focus |
