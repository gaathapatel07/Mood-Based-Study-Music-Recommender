# 🎧 Mood-Based Study Music Recommender

An AI-powered recommendation system that suggests study music based on a user's mood, task type, and energy level.

## 🚀 Features

- Mood-based music recommendations
- Task-specific suggestions
- Energy level analysis
- Machine Learning powered predictions
- Simple and interactive interface

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit

## 📊 Example

### Input

- Mood: Stressed
- Task: Coding
- Energy Level: 4

### Output

- Recommended Music: Ambient Focus
